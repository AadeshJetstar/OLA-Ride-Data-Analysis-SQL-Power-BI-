# 🚖 OLA Ride Data Analysis (SQL + Power BI)

## 📌 Overview

This project analyzes ride booking data using SQL and Power BI to generate meaningful business insights related to bookings, cancellations, and customer behavior.

---

## 🛠 Tools Used

* SQL (MySQL)
* Power BI
* Excel

---

## 🔍 SQL Analysis

* Successful bookings
* Average ride distance
* Cancelled rides
* Top customers
* Driver performance
* Payment analysis
* Ratings analysis

👉 All queries are available in the `sql/queries.sql` file.

---

## 📊 Power BI Dashboard

The dashboard provides insights into:

* Ride volume over time
* Booking status breakdown
* Revenue analysis
* Customer & driver ratings
* Cancellation reasons
